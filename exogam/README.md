This is the EXOGAM detector plugin for the [nptool v.4](https://gitlab.in2p3.fr/nptool/nptool).

Based on the EXOGAM class in [nptool v.3](https://gitlab.in2p3.fr/np/nptool/-/tree/NPTool.2.dev/NPLib/Detectors/Exogam). Developed by [Valerian Girard Alcindor](https://gitlab.in2p3.fr/valerian.girard-alcindor) and [Charlie Paxman](https://gitlab.in2p3.fr/charlie.paxman).

Classes
- `ExogamDetector` = core class, handling treatment of raw and physical events.
- `ExogamSpectra` = nponline data visualisation class, defining spectra and canvases for ROOT interface
- `ExogamData` = raw data class
- `ExogamPhysics` = calibrated data class
